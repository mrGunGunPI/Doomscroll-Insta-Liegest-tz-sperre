# Doomscroll-Insta-Liegest-tz-sperre
Dieses repo gehört zu einer Handy-App aus dem Google Play Store, die die nutzung von instagramm mit einem Liegestütz overlay blockiert, bis eine anzahl von liegestütz gemacht  wurden.


ALEITUNG:
wenn sich das overlav öffnet, bringe dich in eine Liegestütz position, in der das handy dich vollständig von vorn sehen kann. nach 0,5s wird "Bereit" angezeigt. mache nun eine Liegestütze mit dem gesicht zum Handy. Es wird die Statusmeldung "Unten" und "Oben" angezeigt. Eine Liegestütze wird gezählt. Warte nun erneut 0,5s in der Position, bis "Bereit" angezeigt wird.
